# 管理員使用指南（USER_GUIDE_ADMIN）

修玄宮線上候位系統 — 管理後台操作手冊。

## 登入
- 後台登入頁：`/admin/login`
- 預設：帳號 `admin`、密碼 `<ADMIN_PASSWORD>`（由系統管理員設定）
- 建議登入後使用「變更密碼」功能手動更新密碼
- 新增管理帳號：僅現有管理員可建立（一般帳號無法自行註冊或提權）

## 候位管理
- **查看 / 搜尋**：候位列表顯示叫號順序、號碼、狀態、姓名、電話
- **叫號下一位**：自動把順序 1 標記為「處理中」並前移序列
- **拖曳排序**：直接拖動調整叫號順序
- **標記完成**：兩種方式皆可——① 候位列表該筆的「完成」核取方框；② 點「查看詳細資料」→「標記為已完成」
- **詳細資料**：可編輯姓名/電話/地址/家人/諮詢主題/備註
- **取消 / 刪除 / 復原**：皆有確認對話框；取消的記錄可在「已取消客戶」分頁復原

## 結束本期（歸檔）
- 「結束本期」會把本期所有候位記錄（不含已取消）**歸檔到永久客戶資料庫**，並清空候位列表、重設叫號計數
- 操作不可撤銷，有二次確認對話框；送出後按鈕自動鎖定，避免重複點擊造成重複歸檔
- 歸檔時系統會自動比對既有客戶（姓名 + 農曆生日 + 電話等加權判斷）：高信心自動併入同一人、無法判斷者標記為「疑似重複」待人工複核
- 結束本期後下一期報名號碼會從 1 重新開始

## 客戶資料庫
- 「客戶資料庫」可查詢、編輯歷史客戶資料與造訪紀錄（VisitRecord）
- 支援依姓名 / 電話搜尋、查看造訪次數與首末次造訪日期
- **家庭分組**：系統依地址自動將同住客戶歸為一個家庭；可手動「重建家庭分組」重新計算

## 疑似重複客戶複核
- 側欄「疑似重複客戶」列出結束本期歸檔時系統判定「疑似同一人但無法自動確認」的客戶
- 每筆顯示該客戶與疑似對象的姓名/電話/造訪次數/相似度
- 逐筆操作：**「確認同人，合併」**（把兩筆併為一人、造訪紀錄與次數合併）或 **「確認不同人」**（解除標記）
- 合併前系統會自動備份，可事後復原

## 系統設定
- **辦事狀態**：開始 / 停止（僅影響叫號顯示，不影響候位登記開關）
- **最大候位上限**（max-order-index）：控制本期可發出的號碼上限
- **每位客戶預估處理時間**：用於計算前台等待時間
- **簡化模式**：開啟後後台快速登記僅需姓名
- **公開候位登記**：控制一般民眾是否可前台 `/register` 登記
- **查詢顯示號碼**：控制前台查詢是否顯示候位號碼

## 定時開放
- **排程開放時間**：設定下一期自動開放報名的時間
- **自動開放**：開啟後系統到設定時間自動開放公開報名

## 活動橫幅
- 「活動橫幅」可設定前台首頁顯示的公告橫幅（文字 + 按鈕連結）
- 啟用時需填寫有效的按鈕連結，否則橫幅不會顯示

## 資料備份
- **Google Drive 備份**：可手動觸發雲端備份，並查看備份日誌
- **備份列表 / 還原**：列出歷史備份快照，必要時可選一份還原
- 高風險操作（清空候位、合併客戶）都會自動先建快照，可事後復原

## 系統日誌
- 「系統管理」可查看操作日誌，追蹤管理動作與系統事件

## 匯出
- 匯出 Excel / CSV（含所有欄位、中文支援、家人分行）
- 匯出 PDF 問事單（A4 橫式雙 A5）

## 維運工具（進階）
- **重算計數**：候位計數異常時可重新校正
- **還原候位記錄**：從備份還原誤刪的候位（需謹慎）

## 常見問題
- **看不到登記入口**：檢查「公開候位登記」是否開啟，或是否已登入管理員
- **叫號不同步**：刷新首頁或後台；檢查後端 `/ready` 是否回 200
- **結束本期後號碼沒歸零**：已修正（重設時一併歸零發號計數）；若仍異常，用「重算計數」校正
- **同一人被分成兩筆**：到「疑似重複客戶」複核合併；系統已加強自動比對（容忍電話打錯一碼等情況）
